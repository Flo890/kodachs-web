### Usage

* put your API key into secret.js
* run `npm install` and `npm start` / or use `nodemon`
* chat with the Kodachs at `http://localhost:3000`

This requires an active MongoDB server running locally with the default logins and a database called `kodachs-logging`.


### Special Commands

To perform some special operations, the Kodachs reacts to a couple of magic spells:

* `obliviate`: Resets the message history of the current user
